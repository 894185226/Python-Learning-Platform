# Python 变量专题学习网站

## 一、项目简介

本项目是一个面向中学生的 Python 变量专题互动学习网站，采用 **B/S 架构**，集成了以下技术：

| 技术层次 | 技术选型 |
|----------|----------|
| 前端 | HTML5 + CSS3 + JavaScript（W3.CSS 框架） |
| 后端 | Node.js + Express |
| 数据库 | MySQL 8.0 |
| 版本管理 | Git + Gitee |

网站包含 **10 个学习模块**，涵盖情境导入、知识讲解、实践操作、错误调试、课堂小测等教学环节，并内置 **8 项成就系统**，通过闯关激励机制提升学生学习兴趣。

## 三、数据库初始化

### 3.1 创建数据库

打开 MySQL 命令行，执行以下命令：

```sql
CREATE DATABASE IF NOT EXISTS python_var_lesson CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE python_var_lesson;
```

### 3.2 创建数据表

执行以下 SQL 语句（也可直接运行项目中的 `database.sql` 脚本）：

```sql
CREATE TABLE IF NOT EXISTS students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    display_name VARCHAR(50) NOT NULL,
    class_name VARCHAR(50) DEFAULT '',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_username (username)
) ENGINE=InnoDB;

CREATE TABLE IF NOT EXISTS learning_progress (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    module_id VARCHAR(30) NOT NULL,
    completed TINYINT(1) DEFAULT 1,
    score INT DEFAULT 0,
    completed_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    UNIQUE KEY uk_student_module (student_id, module_id)
) ENGINE=InnoDB;

CREATE TABLE IF NOT EXISTS achievements (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    achievement_id VARCHAR(30) NOT NULL,
    earned_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    UNIQUE KEY uk_student_ach (student_id, achievement_id)
) ENGINE=InnoDB;

CREATE TABLE IF NOT EXISTS login_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    login_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    ip_address VARCHAR(45) DEFAULT '',
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    INDEX idx_student_date (student_id, login_time)
) ENGINE=InnoDB;
```

## 四、启动网站

### 4.1 安装依赖

首次使用需安装 Node.js 依赖包：

```bash
cd 项目目录
npm install
```

### 4.2 启动服务器

双击项目中的 **`start_server.bat`** 启动后端服务器。

服务器启动后，命令行窗口显示：

```
Python Variable Adventure - Node.js Server
Starting Node.js server with MySQL database...
Website: http://localhost:3000
```

### 4.3 访问网站

打开浏览器，访问：**http://localhost:3000**

## 五、网站功能模块

### 5.1 学习模块（共 10 个）

| 编号 | 模块名称 | 功能描述 |
|------|----------|----------|
| 1 | 情境导入 | 通过动画场景引入变量概念，激发学习兴趣 |
| 2 | 知识讲解 | 系统讲解变量的定义、赋值与使用方法 |
| 3 | 生活类比实验室 | 用生活中的容器类比变量，直观理解概念 |
| 4 | 命名小法官 | 闯关判断变量命名是否合法，巩固命名规则 |
| 5 | 实践操作 | 分层闯关，动手编写代码创建和使用变量 |
| 6 | 值追踪挑战 | 逐步追踪变量值的变化过程，培养调试思维 |
| 7 | 错误调试诊所 | 找出并修复代码中的错误，锻炼排错能力 |
| 8 | 拓展延伸 | 了解变量的更多应用场景 |
| 9 | 创意项目 | 综合运用知识，完成创意名片生成任务 |
| 10 | 课堂小测 | 检验学习成果，含客观题与能力雷达图 |

### 5.2 成就系统

学生完成学习任务可获得成就徽章：

| 成就 | 图标 | 获得条件 |
|------|------|----------|
| 入门之星 | ⭐ | 完成情境导入和知识讲解 |
| 实验先锋 | 🧪 | 完成生活类比实验室 |
| 公正小法官 | ⚖️ | 完成命名小法官闯关 |
| 编程新星 | 💻 | 完成实践操作模块 |
| 追踪大师 | 🔍 | 完成值追踪挑战 |
| 调试能手 | 🔧 | 完成错误调试诊所 |
| 创意达人 | 🎨 | 完成创意项目 |
| 全能学霸 | 🏆 | 完成全部10个模块 |

### 5.3 成就墙

展示学生学习数据的专门页面，包含：
- **探险地图**：10 个模块完成状态
- **进度条**：模块完成百分比
- **成就徽章墙**：已获得/未获得成就展示
- **学习统计**：完成模块数、成就数、学习天数

## 七、技术亮点

1. **交互式学习体验**：打字机动画、闯关机制、动态反馈
2. **滚动显示效果**：模块滑动进入，增强视觉层次感
3. **成就激励系统**：8 项成就徽章，激发学习动力
4. **数据持久化**：MySQL 存储学生账号与学习记录
5. **密码安全**：SHA256 哈希加密存储
6. **响应式设计**：适配不同屏幕尺寸
