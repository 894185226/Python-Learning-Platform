# Python 基础学习平台 — 使用手册（小白友好版）

## 零、写在前面

本手册会手把手教你如何在任何一台 Windows 电脑上把这个网站跑起来，**不需要任何编程基础**。

> **把文件夹复制到新电脑 → 双击一个文件 → 等待自动完成 → 浏览器自动打开网站**

如果遇到问题，请翻到第九节「常见问题」。

## 一、项目简介

面向中学生的 **Python 基础互动学习平台**，覆盖 Python 入门到进阶的完整知识体系。

| 技术 | 选型 |
|------|------|
| 前端 | HTML5 + CSS3 + JavaScript |
| 后端 | Node.js + Express |
| 数据库 | MySQL |
| 启动 | 双击 `start_server.bat` |

网站包含 **19 个章节，共 139 个学习模块**，内置 **24 项成就系统**。

## 二、快速开始

### 日常启动（每次上课用）

> **双击 `start_server.bat`**

启动后显示：
- 教师机访问：http://localhost:3000
- 管理后台：http://localhost:3000/admin.html（admin / admin123）
- 学生访问：把局域网地址发给学生

### 首次部署（新电脑第一次用）

> **双击 `setup_server.bat`**

自动安装 Node.js、MySQL、npm 依赖，初始化数据库。约 5-15 分钟。

## 三、电脑室局域网部署

1. 教师机双击 `start_server.bat`
2. 记下局域网地址（如 `http://10.20.30.15:3000`）
3. 学生在浏览器输入该地址

## 六、常见问题

**Q: 双击闪退？** → 右键以管理员身份运行，或先运行 setup_server.bat
**Q: 窗口乱码？** → 不影响功能，忽略即可
**Q: 无法登录？** → 确认 MySQL 服务已启动（services.msc）
**Q: 学生无法访问？** → 检查是否同一局域网，防火墙是否放行 3000 端口
**Q: 端口被占用？** → `netstat -ano | findstr :3000` 然后 `taskkill /f /pid xxx`
**Q: .ps1 是什么？** → PowerShell 脚本，不需要直接双击，双击 .bat 即可
**Q: 两个 .bat 区别？** → setup_server.bat 首次部署用，start_server.bat 日常启动用
