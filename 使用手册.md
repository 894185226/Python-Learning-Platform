# Python 变量专题学习网站 - 使用手册

## 一、环境准备

### 1.1 所需软件

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| MySQL | 8.0+ | 用于存储学生账号和学习数据 |
| Node.js | 16+ | 运行后端服务器 |

### 1.2 安装 MySQL

1. 前往 [MySQL 官网](https://dev.mysql.com/downloads/mysql/) 下载安装包
2. 安装时选择「Full」安装类型
3. 设置 root 用户密码，**请记住此密码**
4. 安装完成后，MySQL 服务会自动启动

### 1.3 安装 Node.js

1. 前往 [Node.js 官网](https://nodejs.org/) 下载 LTS 版本
2. 双击安装包，一路点击「Next」即可
3. 安装完成后，打开命令提示符，输入 `node -v` 验证安装成功

## 二、初始化数据库

### 2.1 创建数据库

1. 打开 MySQL 命令行客户端（或使用 Navicat 连接）
2. 执行以下命令创建数据库：

```sql
CREATE DATABASE IF NOT EXISTS python_var_lesson CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE python_var_lesson;
```

### 2.2 创建数据表

依次执行以下 SQL 语句创建四张表：

```sql
-- 学生用户表
CREATE TABLE IF NOT EXISTS students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    display_name VARCHAR(50) NOT NULL,
    class_name VARCHAR(50) DEFAULT '',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_username (username)
) ENGINE=InnoDB;

-- 学习进度表
CREATE TABLE IF NOT EXISTS learning_progress (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    module_id VARCHAR(30) NOT NULL,
    completed TINYINT(1) DEFAULT 1,
    score INT DEFAULT 0,
    completed_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    UNIQUE KEY uk_student_module (student_id, module_id)
) ENGINE=InnoDB;

-- 成就记录表
CREATE TABLE IF NOT EXISTS achievements (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    achievement_id VARCHAR(30) NOT NULL,
    earned_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    UNIQUE KEY uk_student_ach (student_id, achievement_id)
) ENGINE=InnoDB;

-- 登录日志表
CREATE TABLE IF NOT EXISTS login_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    login_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    ip_address VARCHAR(45) DEFAULT '',
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    INDEX idx_student_date (student_id, login_time)
) ENGINE=InnoDB;
```

> 也可以直接运行项目中的 `database.sql` 脚本文件完成上述操作。

## 三、启动网站

### 3.1 启动后端服务器

双击项目文件夹中的 **`start_server.bat`** 文件

或手动在命令行中执行：

```bash
cd 项目目录
node server.js
```

### 3.2 访问网站

打开浏览器，访问：**http://localhost:3000**

## 五、使用 Navicat 查看数据

### 5.1 新建连接

1. 打开 Navicat
2. 点击「连接」→「MySQL」
3. 填写连接信息：
   - 连接名：任意名称（如「Python变量学习」）
   - 主机：localhost
   - 端口：3306
   - 用户名：root
   - 密码：你的 MySQL root 密码

### 5.2 查看数据

连接成功后，双击数据库 `python_var_lesson`，即可查看：
- `students` — 学生账号信息
- `learning_progress` — 模块学习进度
- `achievements` — 成就获得记录
- `login_logs` — 登录日志

## 六、常见问题

### Q1：启动服务器提示「node 不是内部或外部命令」

确保 Node.js 已正确安装，并将 Node.js 添加到系统 PATH 环境变量。

### Q2：访问网站提示「无法连接服务器」

检查后端服务器是否正在运行，命令行窗口应保持打开状态。

### Q3：注册/登录失败

检查 MySQL 服务是否正常运行，以及数据库是否已正确创建。

### Q4：浏览器显示乱码

确保 MySQL 连接使用 `utf8mb4` 字符集。

## 八、技术架构

