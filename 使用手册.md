# Python 变量专题学习网站

## 一、项目简介

本项目是一个面向中学生的 Python 变量专题互动学习网站，采用 **B/S 架构**，集成了以下技术：

| 技术层次 | 技术选型 |
|----------|----------|
| 前端 | HTML5 + CSS3 + JavaScript（W3.CSS 框架） |
| 后端 | Node.js + Express |
| 数据库 | MySQL 8.0 |
| 版本管理 | Git + Gitee |

网站包含 **10 个学习模块**，涵盖情境导入、知识讲解、实践操作、错误调试、课堂小测等教学环节，并内置 **8 项成就系统**，通过闯关激励机制提升学生学习兴趣。获得成就时，页面右下角会弹出 Steam 风格的成就提示框。

## 三、数据库初始化

### 3.1 创建数据库

打开 MySQL 命令行，执行以下命令：

```sql
CREATE DATABASE IF NOT EXISTS python_var_lesson CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE python_var_lesson;
```

### 3.2 创建数据表

执行以下 SQL 语句（也可直接运行项目中的 `database.sql` 脚本）：

```sql
CREATE TABLE IF NOT EXISTS students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    display_name VARCHAR(50) NOT NULL,
    class_name VARCHAR(50) DEFAULT '',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_username (username)
) ENGINE=InnoDB;

CREATE TABLE IF NOT EXISTS learning_progress (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    module_id VARCHAR(30) NOT NULL,
    completed TINYINT(1) DEFAULT 1,
    score INT DEFAULT 0,
    completed_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    UNIQUE KEY uk_student_module (student_id, module_id)
) ENGINE=InnoDB;

CREATE TABLE IF NOT EXISTS achievements (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    achievement_id VARCHAR(30) NOT NULL,
    earned_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    UNIQUE KEY uk_student_ach (student_id, achievement_id)
) ENGINE=InnoDB;

CREATE TABLE IF NOT EXISTS login_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    student_id INT NOT NULL,
    login_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    ip_address VARCHAR(45) DEFAULT '',
    FOREIGN KEY (student_id) REFERENCES students(id) ON DELETE CASCADE,
    INDEX idx_student_date (student_id, login_time)
) ENGINE=InnoDB;
```

## 四、启动网站

### 4.1 安装依赖

首次使用需安装 Node.js 依赖包：

```bash
cd 项目目录
npm install
```

### 4.2 启动服务器

双击项目中的 **`start_server.bat`** 启动后端服务器。

> 脚本会自动检测 Node.js 路径并安装依赖，支持跨电脑移植。

服务器启动后，命令行窗口显示：

```
Python Variable Adventure - Node.js Server
Starting Node.js server with MySQL database...
Website: http://localhost:3000
```

### 4.3 访问网站

打开浏览器，访问：**http://localhost:3000**

| 追踪大师 | 🔍 | 完成值追踪挑战 |

| 创意达人 | 🎨 | 完成创意项目 |
| 全能学霸 | 🏆 | 完成全部 10 个模块 |

### 5.3 成就墙（回顾与成就墙页面）

展示学生学习数据的专门页面，包含：

- **探险地图**：10 个模块完成状态，已完成模块以绿色标记
- **进度条**：模块完成百分比
- **成就徽章墙**：已获得/未获得成就展示，含获得日期
- **学习统计**：完成模块数、成就数、学习天数

### 5.4 搜索功能

导航栏及首页均设有搜索框，支持智能模糊搜索：

- **实时建议**：输入时实时弹出匹配的模块建议列表
- **关键词匹配**：支持中文关键词搜索（如输入"代码" → 实践操作，"bug" → 错误调试诊所）
- **模糊匹配**：即使输入有错别字也能匹配到对应模块

### 5.5 访问控制

- **未登录用户**：仅可查看首页基础介绍，点击具体模块时弹出登录窗口
- **已登录用户**：完整访问所有学习模块，退出登录后自动刷新页面

## 十、常见问题

### Q1：双击 start_server.bat 后闪退？
请确认已安装 Node.js。可在命令行输入 `node -v` 验证安装。

### Q2：登录时显示"服务器错误"？
请确认 MySQL 服务已启动，且 `python_var_lesson` 数据库已创建。

### Q3：数据库中文乱码？
创建数据库时需指定 `CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci`。

### Q4：如何查看注册账号的数据？
可使用 Navicat 等工具连接本地 MySQL，查询 `python_var_lesson` 数据库中的 `students`、`learning_progress`、`achievements` 表。

### Q5：如何重置某位学生的数据？
在 MySQL 中执行（以用户名为 `测试1` 为例）：

```sql
SET @sid = (SELECT id FROM python_var_lesson.students WHERE username = '测试1');
DELETE FROM python_var_lesson.learning_progress WHERE student_id = @sid;
DELETE FROM python_var_lesson.achievements WHERE student_id = @sid;
DELETE FROM python_var_lesson.login_logs WHERE student_id = @sid;
DELETE FROM python_var_lesson.students WHERE id = @sid;
```
